# Wearable Smart Alarm Armband (Work in progress)

An Armband designed primaraly for force waking up in the morning using vibration and Electrical stimulation.Uses an android app to communicate via BLE for scheuling and monitering status such as battery capacity.Integrated IMU is used to detect weather the user is awake or sleeping.
